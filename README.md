# Titanic Survival Prediction

This project solves the Kaggle "Titanic: Machine Learning from Disaster" competition
using classical machine learning techniques.

## Problem Statement
Predict whether a passenger survived the Titanic disaster based on demographic
and ticket-related features.

## Dataset
- Source: Kaggle Titanic dataset
- Train size: 891 samples
- Test size: 418 samples

## Approach
- Data cleaning and preprocessing
- Handling missing values
- Feature encoding
- Feature engineering
- Feature scaling
- Logistic Regression model

## Model Performance
- Accuracy: ~71% on Kaggle leaderboard

## Files
- titanic_survival_prediction.ipynb : Main notebook
- submission.csv : Kaggle submission file
